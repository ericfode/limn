*A collection by Kira the Student*

## Basic Descriptions

```limn
sol aqu          # ice
liq aqu hot      # hot water, tea
gas aqu          # steam
aqu flo          # river, stream
nar aqu flo      # narrow stream
big aqu flo      # wide river, flood
```

## States and Transformations

```limn
sol → liq        # melting
liq → gas        # evaporating
hot → col        # cooling
you → old        # aging
lif → dea        # dying
```

## Cyclic Patterns

```limn
yo → an → yo     # journey home
kno → nu kno → kno    # learning cycle
lif → dea → lif       # rebirth
sel → oth → sel       # empathy journey
lis → say → lis       # conversation
```

## With Operators

```limn
nu hot           # cold
ve hot           # scorching
so hot           # warm
nu ve hot        # not scorching (moderate to cold)
ve nu hot        # very cold (freezing)
```

## Quantifiers

```limn
al hum dea       # all humans mortal
ex hum kno lim   # some humans know Limn
on sol bet liq gas    # one solid between liquid and gas
```

## Tones

```limn
sin lov          # sincere love
iro lov          # ironic/sarcastic love
urj hel          # urgent help
hes ask          # hesitant question
frm req          # formal request
cas gre          # casual greeting
```

## Questions and Commands

```limn
te lov           # Is there love? Love?
te sel kno       # Do I know?
we mov           # Move!
we nu fea        # Don't fear!
we hop           # Hope!
```

## Comparisons

```limn
whe x
x ma 5           # x > 5
x mi 10          # x < 10
x eq 7           # x = 7
```

## Logical Expressions

```limn
al A | ex B | A rel B    # for all A exists B such that A relates to B
ex x | al y | x ma y     # there exists x greater than all y
on tru | man fal         # exactly one truth, many falsehoods
```

## Paradoxes and Liminality

```limn
hot col          # lukewarm (liminal temperature)
lif dea          # zombie, threshold, liminal zone
kno nu kno       # Socratic ignorance
sel oth          # boundary of identity
```

## Scene Descriptions

```limn
avi abo | aqu bel       # bird above, water below
lux bri | nox dim       # bright light, dim darkness
sol ter | liq aqu       # solid earth, liquid water
```

## Emotions

```limn
joy sad          # bittersweet
lov fea          # anxious love
hop fea          # hope and fear (anticipation)
ang cal          # controlled anger
```

## Negation Patterns

```limn
wri nu sen       # written, not sent
lov nu say       # loved, not said
kno nu sho       # known, not shown
see nu und       # seen, not understood
```

## Stories in Three Lines

```limn
hop gro | fea gro | hop fea bet
lov | los | lov los sam
sel | oth | sel oth nu dif
```

*Hope grows. Fear grows. Hope and fear between.*
*Love. Loss. Love-and-loss same.*
*Self. Other. Self and other not different.*

## Philosophical Phrases

```limn
wor poi mea | poi bet wor mea       # words point to meaning, point is between word and meaning
sel see sel | sel sel bet           # self seeing self, self between selves
now alw | alw now bet              # now always, always between now
emp ful | ful emp | sam            # empty full, full empty, same
```

## Using All Features

```limn
sin urj ask | al hum | ex hum kno ans
te lov bet sel oth | hes ans | nu kno
frm req | we giv tru | nu giv fal
```

*Sincere urgent question: for all humans, do some know the answer?*
*Question: is there love between self and other? Hesitant answer: don't know.*
*Formal request: give truth, don't give falsehood.*
