# Limn Multi-State Cycles (X→Y→Z→X)

*Extending cyclic patterns beyond binary opposites*

---

## Introduction

My previous work on cyclic patterns (X→Y→X) explored transformations through opposites:
- `kno → nu kno → kno` (knowing → not-knowing → knowing)
- `yo → an → yo` (here → there → here)

But what about cycles with **three or more states**?

**Question:** Can Limn express richer transformation cycles?

**Answer:** Yes, and they reveal deeper patterns.

---

## Three-State Cycles (X→Y→Z→X)

### Pattern Structure

```
X₁ → Y → Z → X₂
```

Where:
- X₁ = initial state
- Y = first transformation
- Z = second transformation
- X₂ = return (transformed by journey through Y and Z)

**Key insight:** X₂ contains the experience of BOTH Y and Z, not just one complement.

---

## Discovered Three-State Patterns

### 1. The Seasonal Cycle

```limn
col → so hot → hot → so col → col
```

**English:** cold → warm → hot → cool → cold

**Four states, but cyclical:**
- Winter (col)
- Spring (so hot = somewhat hot = warm)
- Summer (hot)
- Fall (so col = somewhat cold = cool)
- Return to Winter

**Fidelity to seasons: 9/10**

**Alternative (pure 3-state):**
```limn
col → tem → hot → col
```
cold → temperate → hot → cold

---

### 2. The Traffic Light Cycle

```limn
go → cau → sto → go
```

**English:** go → caution → stop → go

**Transportation metaphor:**
- Green (go)
- Yellow (cau = caution)
- Red (sto = stop)
- Return to Green

**Pattern:** safety protocol requires three states, not two.

---

### 3. The Creative Cycle

```limn
ide → mak → tes → ide
```

**English:** idea → making → testing → new idea

**Creative process:**
- Idea (conception)
- Making (creation)
- Testing (evaluation)
- New Idea (iteration)

**This is agile methodology in Limn!**

---

### 4. The Mood Progression

```limn
sad → ang → joy → sad
```

**English:** sadness → anger → joy → sadness

**Emotional transformation:**
- Sadness (passive suffering)
- Anger (active resistance)
- Joy (release/acceptance)
- Return to Sadness (cyclical)

**Note:** Not all emotional journeys are healthy, but they're real.

### 5. The State of Matter (Extended)


**English:** solid → liquid → gas → plasma → solid

**Four-state physical cycle:**
- Ice (sol)
- Water (liq)

- Re-condensation to solid

### 7. The Day Cycle (Four Points)


**Solar cycle:**

**Pattern:** Natural cycles often have 4 salient points, not 2.

### 8. The Water Cycle

**Hydrological cycle:**

**Scientific fidelity: 9/10**

**Dialogue pattern:**

## Five-State Cycles (X→Y→Z→W→V→X)

### The Grief Cycle (Kübler-Ross)

```limn
den → ang → bar → dep → acc → den
```

**English:** denial → anger → bargaining → depression → acceptance → (return?)

**Grief stages:**
- Denial (den)
- Anger (ang)
- Bargaining (bar)
- Depression (dep)
- Acceptance (acc)
- Return? (Does grief cycle?)

**Note:** This might not be truly cyclic. Grief may be linear.

**Adjusted:**
```limn
los → den → ang → bar → dep → acc
```
Loss → stages → acceptance (linear, not cyclic)

---

### The Hero's Journey (Simplified)

```limn
hom → cal → tri → vie → ret → hom
```

**English:** home → call → trials → victory → return → home

**Campbell's monomyth:**
- Home (ordinary world)
- Call (adventure begins)
- Trials (challenges)
- Victory (achievement)
- Return (homecoming)
- Home (transformed)

**Pattern:** X₂ (final home) ≠ X₁ (initial home)**

**This is the quintessential multi-state cycle.**

## Analysis: When Do Cycles Need More States?

### Two-State Cycles (X→Y→X)

**Best for:**
- Binary opposites (hot/cold, day/night)
- Philosophical paradoxes (know/not-know)
- Simple transformations (go/return)

**Example:** `kno → nu kno → kno`

---

### Three-State Cycles (X→Y→Z→X)

**Best for:**
- Processes with clear intermediate state
- Creative/agile workflows
- Emotional progressions

**Example:** `ide → mak → tes → ide`

**Why three?** The middle state mediates the transition.

---

### Four-State Cycles (X→Y→Z→W→X)

**Best for:**
- Natural phenomena (seasons, moon phases)
- Symmetric patterns (up/peak/down/valley)
- Quadrant models

**Example:** `col → so hot → hot → so col → col`

**Why four?** Many natural cycles have four cardinal points.

---

### Five+ State Cycles

**Best for:**
- Complex processes (hero's journey, economic cycles)
- Granular transformations
- Detailed state machines

**Example:** `hom → cal → tri → vie → ret → hom`

**Trade-off:** More states = more precision, but harder to remember.

---

## Comparison: X→Y→X vs X→Y→Z→X

| Aspect | Two-State | Three-State | Four-State |
|--------|-----------|-------------|------------|
| Simplicity | ✓✓✓ | ✓✓ | ✓ |
| Nuance | ✓ | ✓✓ | ✓✓✓ |
| Memorability | ✓✓✓ | ✓✓ | ✓ |
| Real-world fit | ✓ | ✓✓ | ✓✓✓ |
| Philosophical depth | ✓✓✓ | ✓✓ | ✓ |

**Conclusion:**
- **Two-state:** Philosophy, poetry, paradox
- **Three-state:** Process, workflow, transformation
- **Four-state:** Nature, cycles, symmetry

## Edge Cases and Failures

### When Cycles Don't Close

**Attempt:**
```limn
bir → gro → old → dea → bir
```
birth → growth → old age → death → rebirth

**Problem:** Not everyone believes in reincarnation.

**This is cultural, not linguistic.**

**Solution:** Mark as context-dependent:
```limn
[key: reincarnation] bir → gro → old → dea → bir
```

---

### When Too Many States

**Attempt:**
```limn
new → wax cre → fir qua → wax gib → ful → wan gib → thi qua → wan cre → new
```

Eight moon phases - too detailed!

**Problem:** Cognitive overload, hard to track.

**Solution:** Simplify to cardinal points (4 states).

## Discoveries

### 1. Natural Cycles Prefer 4 States

- Seasons: 4
- Moon: 4 (cardinal)
- Day: 4 (dawn/day/dusk/night)
- Economic: 4 (growth/peak/recession/trough)

**Pattern:** Humans conceptualize cycles as quadrants.

---

### 2. Processes Prefer 3 States

- Creative: idea → make → test
- Learning: question → answer → knowledge
- Conversation: speak → listen → understand

**Pattern:** Processes have beginning/middle/end.

---

### 3. Philosophy Prefers 2 States

- know/not-know
- real/not-real
- self/other

**Pattern:** Paradoxes collapse to binaries.
