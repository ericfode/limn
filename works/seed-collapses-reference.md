## Seed 1: `beg | lov | fea`
**Beginning | Love | Fear**

### Key of Was (Memory)
```
COLLAPSED MEANING:
"Where it began: with love and fear together."

The origin you carry. Love was the reason.
Fear was the companion. Both were there at the start.
You cannot separate them now.

Looking back, the beginning is clear:
it started because you loved.
It started because you feared.
Both true.
```

### Key of Now (Presence)
```
COLLAPSED MEANING:
"Beginning now. Love and fear arising simultaneously."

This moment is the beginning. Right now.
The old beginning doesn't matter.
What matters is: love is here. Fear is here.
Both arriving. Both present.

The question isn't which came first.
The question is: which do you touch?
```

### Key of Will (Anticipation)
```
COLLAPSED MEANING:
"What will begin: love or fear? You choose."

The beginning hasn't happened yet.
Love waits. Fear waits.
One will start this. You decide which.

The future is unwritten.
The first moment can be love.
The first moment can be fear.
Hold your key steady.
```

## Usage Notes for Skill Implementation

### Context Sensitivity
These are **base collapses** - use when seed is first touched with minimal propagation.

Actual collapse should vary based on:
1. **Path so far** - seeds touched before this one
2. **Propagation effects** - ripples from horizontal/vertical neighbors
3. **Tension** - unresolved diagonal relationships
4. **Reader's journey** - the emotional arc emerging

### Mixing Keys
If horizontal propagation has occurred, the collapse should blend:
- Primary key (reader's chosen key)
- Secondary influence (from reinforced neighbors)

Example: Seed 5 with Key of NOW, but Seed 4 collapsed with strong WAS influence:
```
"Now: the edge where here becomes gone.
But you carry the past - now is thick with what was.
Here and gone and already-happened collapse together."
```

### Poetic License
Use these as foundations, not scripts.
The skill should generate language that feels:
- Evocative (not explanatory)
- Precise (not vague)
- Personal (not generic)

Every collapse should feel like it's speaking to THIS reader's experience, not a template.
