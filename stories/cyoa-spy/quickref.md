# Quick Reference - Spy Limn

## Agents
```
car = Cardinal (captured operative)
roo = Rook (new agent)
bis = Bishop (experienced, suspect)
kni = Knight (retired, local)
arc = Architect (enemy spymaster)
han = Handler (you)
```

## Status
```
saf = safe
tak = taken/captured
ali = alive
dea = dead
mis = missing
bur = burned (cover blown)
```

## Actions
```
mov = move
sta = stay
ext = extract
ent = enter
fol = follow
wai = wait
run = run
```

## Locations
```
loc = location
bui = building
lev = level
roo = room
ent = entrance
beh = behind/rear
fro = front
```

## Time
```
tim = time
urj = urgent
sho = short
lon = long
now = now
soo = soon
```

## Trust
```
tru = trust
dou = doubt
loy = loyal
tra = traitor/trap
tes = test
pro = prove
```

## Tone Markers
```
frm = formal (official orders)
cas = casual
sin = sincere (honest)
iro = ironic (inverted meaning)
hes = hesitant (possibly coerced)
urj = urgent (!)
te = question (?)
we = imperative (!)
```

## Common Patterns
```
[agent] | [status] | [location] | [action]
car tak | loc unk | tim sho | urj
= Cardinal taken, location unknown, time short, urgent

[tone] | [message]
frm con | mov aut
= Formal: confirmed, movement authorized

[question]
te ord = orders?
te acc = accept?
te sta = status?
```

## Keys
```
sur = surface (innocent reading)
cov = cover (allied services)
tru = true (network meaning)
gho = ghost (sender/recipient only)
```
