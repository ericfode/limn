# Daily Oracle

```limn
lux dim lim
```
*twilight. the boundary between light and dim.*

Neither day nor night.
The boundary where both are true.

Dawn holds the memory of darkness.
Dusk holds the promise of stars.

You stand in neither.
You stand in both.
