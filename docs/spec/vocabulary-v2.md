# Limn Vocabulary v2 - Comprehensive

**Design Goal:** Maximize expressive power within 100k token bootstrap budget.

**Strategy:**
- ~500 core words organized by semantic domain
- Each word = CV or CVC syllable
- Words chosen for maximum constraint diversity
- Domains cover: physical, temporal, spatial, relational, cognitive, social, abstract

## Usage Examples

### Simple Sentences
```
sol aqu dur = solid water ongoing = ice, glacier, persistent moisture
lif gro you = alive growth young = seedling, child, new organism
thi que ans = thinking question answer = problem-solving, dialogue
```

### With Operators
```
nu sol = not-solid = liquid, gas, or non-physical
ve lux = very-bright = sun, explosion, enlightenment
yo fea an hop = this-fear that-hope = present anxiety, future hope
```

### Complex Combinations
```
hum lif gro joi com = human alive growth join community = civilization rising
sol flo pat blo end = solid flow path block end = dam, barrier, conclusion
thi cre new tra mod = thinking create new tradition modify = cultural evolution
```

### With Scope
```
(sol | liq) hot = (solid or liquid) hot = molten material, hot water
nu (fea des) = not (fear despair) = courage, hope
yo > an = this to that = movement, change, gift
```
