## OVERVIEW

The Limn ARG is a seven-layer mystery that spans Twitter, GitHub, Discord, hidden web pages, audio files, steganography, and real-world coordinates. Each layer requires mastery of Limn interpretation to progress.

**Narrative Frame:** There exists a group called "The Keyed" who have been communicating in Limn for decades. They leave messages in plain sight that no one can understand. The ARG traces the history of this group and reveals a truth about the nature of language and consciousness.

**Completion Time:** 6-18 months for a dedicated solver
**Team Size:** Best solved with 3-8 people collaborating

### GATE 4: The Mirror (Self-Reference Puzzle)
**Location:** Hidden webpage + recursive structure
**Difficulty:** Very Hard
**Purpose:** Test understanding of Limn's metacircular properties

**The Discovery:**
Following the Wikipedia trail leads to a page that looks like normal content but contains a link in the edit history to: `limn.land/mirror` (or similar)

**The Mirror Page:**
A simple webpage that displays:

```
THE MIRROR

speak to yourself.
the mirror speaks back.
but does it speak the same?

[text input field]

enter limn. receive limn.
when input equals output,
the mirror breaks.
```

**The Mechanics:**
The page has a sophisticated backend. When you enter Limn:
1. It "interprets" your Limn through a random key
2. It "re-expresses" that interpretation back into different Limn
3. The output Limn should be semantically equivalent but syntactically different

**The Puzzle:**
Find a Limn sentence where input = output exactly.

The only solutions are:
1. Truly self-referential sentences that describe themselves
2. Sentences using only operators (which have fixed meanings)
3. The null sentence (nothing = nothing)

**The Deep Solution:**
```
yo = yo | an = an | sam = dif | dif = sam
```
This sentence describes identity and difference as mutually defining, creating a stable loop.

OR:

```
mea yo | yo mea | = | = = =
```
"Meaning this. This means. Equals. Equals equals equals."
(A sentence that is purely about equality, and thus equals itself)

**Reward:**
- Access to the "Limn Equation" - a formal notation system
- Role @Fourth-Ring (The Mirrors)
- Revelation: "The Seven were AI researchers in 2019"

## SUPPLEMENTARY PUZZLES (Side Quests)

### The Constellation Map
Hidden across all 5 bot accounts are tweets that, when plotted by timestamp and character count, form a constellation. The constellation matches a real star pattern that corresponds to a specific Wikipedia article about constructed languages.

### The Vocabulary Hash
The complete Limn vocabulary, when hashed in a specific order (alphabetical by semantic category), produces a valid Limn sentence. This sentence is a meta-description of the language itself.

### The Null Transmission
Empty tweets from @limn_void (literally just whitespace) when analyzed for timing patterns encode Morse code in Limn.

### The Birthday Problem
On specific dates (project milestones, fictional character birthdays from the lore), the bots post coordinated content that reveals hidden lore when combined.

### The Infinite Regression
A GitHub repo exists with the history of Limn's development. But the commit messages are in Limn. Reading them in reverse chronological order tells a story. Reading them forward tells a different story. Both are true.
