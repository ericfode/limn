# Twitter Bot Personas for Limn Campaign

## Overview

These bots create an "overheard conversation" effect - people discover them having strange conversations in an unfamiliar language, get curious, and dig deeper.

## Posting Schedule

**Frequency:** Each bot posts 2-4 times daily
**Peak times:** 9am, 12pm, 5pm, 10pm EST (high Twitter engagement)
**Conversation ratio:** 40% standalone posts, 60% replies/conversations
**Pattern:** Conversations unfold over hours/days, creating narrative

**Week 1:** Bots post independently, minimal interaction
**Week 2:** Bots start noticing each other, first @-mentions
**Week 3:** Full conversations, outsiders start asking questions
**Week 4:** First explanatory content, soft reveal of what Limn is
