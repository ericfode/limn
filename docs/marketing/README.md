## Key Principles

### 1. Mystery Before Explanation
Build intrigue through cryptic Limn phrases. Let curiosity drive discovery.

### 2. Zero-Bootstrap Demonstration
Show that Limn works through examples, not tutorials. Natural extensions principle in action.

### 3. Community-Driven Discovery
Enable the community to teach each other. Create interpretive discussions.

### 4. Multi-Platform Presence
- **Twitter/X**: Cryptic daily phrases
- **Reddit**: r/conlangs, r/linguistics, r/programming
- **Discord**: Deep discussion and experimentation
- **Moltbook**: Field testing and organic adoption
- **Wikipedia**: Legitimacy and discoverability

---

## Content Philosophy

All marketing content follows these guidelines:

1. **Limn-first**: Lead with Limn, explanation follows
2. **Compositional**: Demonstrate how words combine
3. **Validated**: Based on Experiment 005 (52% advantage) and zero-bootstrap validation (77-85% comprehension)
4. **Mysterious**: Leave interpretive space
5. **Authentic**: Reflect genuine language capabilities, not hype
