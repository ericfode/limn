## Emergency Playbook

### If engagement is too low
- Increase posting frequency
- Create more provocative/viral content
- Cross-promote on adjacent communities (conlang, linguistics, AI)

### If harassment occurs
- Bots do not engage with trolls
- Discord moderation active
- Block/mute as needed

### If the puzzle is too hard
- Release hints per schedule
- Monitor frustration levels
- Adjust difficulty of future puzzles

### If someone reverse-engineers everything
- Celebrate them publicly
- Invite them to be a moderator
- This is success, not failure
