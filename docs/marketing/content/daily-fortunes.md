# Daily Limn Fortunes

*A collection of fortune-cookie style Limn sentences with interpretations*
*For automated daily posting or @limn_fortune bot*

---

## Format

Each fortune is:
1. A Limn sentence (3-5 words)
2. A cryptic interpretation (2-4 lines)
3. Multiple valid readings

## Usage Notes

### For Daily Posting
- Post at a consistent time (suggest: 7am local)
- No explanation beyond the fortune itself
- Let readers interpret
- Occasional reply with "key?" to engagement

### For @limn_fortune Bot
- Automated daily selection
- Random from pool
- Never repeats within 30 days
- Tracks which fortunes get most engagement

### Expansion
- Add fortunes as they're written
- Community can submit via Discord
- Best community submissions added to official pool
