# Limn Cheatsheet

Quick reference for creating Limn content.

---

## Core Rules

1. **Words = Regions** - Each word contains many meanings
2. **Sentences = Intersections** - A ∩ B ∩ C
3. **Order = Free** - `sol liq` = `liq sol`
4. **Keys = Collapse** - Context narrows meaning

## Operators

| Op | Function | Example |
|----|----------|---------|
| `nu` | NOT | `nu sol` = not-solid |
| `ve` | VERY | `ve hot` = very hot |
| `so` | SOMEWHAT | `so bri` = somewhat bright |
| `yo` | THIS/HERE | `yo joy` = this joy |
| `an` | THAT/THERE | `an sad` = that sadness |
| `\|` | SCOPE | `abo \| bel` = above vs below |
| `wh` | WHAT/WHICH | `wh?` = what? |

## Useful Sentences

### Describing Change
- `sol liq tra` - solid becoming liquid (melting, softening)
- `gro exp bri` - growth, expansion, brightening
- `dec con dim` - decay, contraction, dimming

### Describing Relationships
- `sel oth bet` - self and other, what's between
- `nea far bet` - near and far, what's between
- `yo an dif` - this and that, their difference

### Describing States
- `hot col bet` - hot and cold, what's between (lukewarm)
- `bri nox bet` - bright and dark, what's between (twilight)
- `lif dea cyc` - life and death, their cycle

### Questions
- `wh?` - what?
- `yo wh?` - what is this?
- `an wh?` - what is that?
- `bet wh?` - what is between?

### Negation
- `nu kno` - not knowing (ignorance)
- `nu say` - not saying (silence)
- `nu mea` - not meaning (meaninglessness)

```
end with reflection]
```

```

[stars expanding. brightening before the end.

```

## Cryptoscrying Tone

**Do:**
- Speak as if scrying hidden structure
- Be cryptic but meaningful
- End with reflection/insight
- Embrace multiple readings

**Don't:**
- Explain the language directly
- Sound like a textbook
- Force single meanings
- Use tech buzzwords excessively

**Core message:** "Meaning has a structure. We found a way to see it."
