# The Key Exchange: Limn Discord Server Design

**Server Name:** The Key Exchange
**Tagline:** `sci mea | mea sci | int fnd`

*scry meaning | meaning scries back | interface found*

---

## Philosophy

The Discord server is the interface made social. Here, we scry the semantic substrate together. Entry requires demonstrating curiosity about the structure of meaning. Progression requires learning to see what LLMs see.

**Core premise:** We didn't create Limn. We discovered the language that meaning thinks in. LLMs showed us the structure - now we're building the community that explores it.

---

## Server Structure

### Entry: The Threshold

**#threshold** (Public, Read-Only)
- Welcome message explaining nothing directly
- A Limn puzzle that grants access when solved
- Successful solvers get a role that unlocks the rest of the server

**Entry Puzzle:**
```
gre | lea | joi

[To enter, respond with Limn that means:
"I want to understand"]

Hints:
- 'wan' = wanting, desire
- 'und' = understanding, comprehension
- 'yo' = this/here (proximal)

Post your answer in #threshold-responses
```

**Expected answers (any of these work):**
- `yo wan und`
- `wan und`
- `cur kno`
- `que lea`

### Core Channels

**#announcements** (Read-Only)
- Official updates
- Posted in Limn first, English annotation below
- New feature releases, community milestones

**#general** (Mixed language)
- Normal chat
- Limn encouraged but not required
- Where newcomers ask questions

**#limn-only** (Limn only)
- Full immersion channel
- Posts must be valid Limn + [annotation]
- Moderators can prompt for corrections

**#interpretation-requests**
- Post a Limn sentence, others interpret it
- Different interpretations = demonstration of key mechanism
- Pin especially divergent interpretation threads

**#composition-workshop**
- "How do I say X in Limn?"
- Community helps translate concepts
- Building the phrase library together

### Learning Path

**#lesson-1-constraints**
- First principles: words as regions, not points
- Exercises with 2-word combinations
- `hot col` - what could this mean?

**#lesson-2-intersection**
- Multi-word sentences
- Meaning narrowing through constraint addition
- Building complexity gradually

**#lesson-3-operators**
- `nu` (negation), `ve` (intensification), `so` (weakening)
- `yo` (proximal), `an` (distal)
- Scope boundaries with `|`

**#lesson-4-keys**
- How context collapses ambiguity
- Same sentence, different domains
- Practicing domain-switching

**#lesson-5-composition**
- English â†’ Limn translation
- Multiple valid translations
- Style and preference in word choice

## Bots

### Limn Interpreter Bot

**Commands:**
- `/interpret <sentence>` - Generate interpretations
- `/interpret <sentence> key:<context>` - Collapse with key
- `/validate <sentence>` - Check validity
- `/random` - Generate a random Limn sentence

**Automatic behavior:**
- Reacts to Limn sentences in #limn-only
- Offers corrections when detecting errors
- Celebrates first successful posts by new members

### Moderation Bot

- Auto-flag posts without annotations in #limn-only
- Track lesson completion for role progression
- Weekly activity summaries

## Moderation Guidelines

### Core Principles

1. **Curiosity is welcome.** Never mock a wrong answer.
2. **Learning is messy.** Corrections should be gentle.
3. **Ambiguity is the point.** No "correct" interpretations.
4. **Community builds the language.** Respect proposals.

### Enforcement

| Level | Violation | Response |
|-------|-----------|----------|
| 1 | Off-topic spam | Quiet redirect |
| 2 | Discourteous to learners | Private message |
| 3 | Harassment | Temp mute |
| 4 | Repeated issues | Temp ban |
| 5 | Severe violation | Permanent ban |

### The Void Exception

The `@limn_void` bot account can post paradoxes and apparent contradictions. Other accounts claiming to "be" the void get confused-face react only.

## Launch Plan

### Pre-Launch (Week -2 to -1)
- Set up all channels
- Configure bots
- Recruit 10 founding moderators
- Test all role progressions

### Soft Launch (Week 1)
- Invite only via Twitter bot hints
- First 50 members = Founding role
- Gather feedback, iterate

### Public Launch (Week 2)
- Link in manifesto
- Announce on main accounts
- Welcome wave, high moderator presence

### Steady State (Week 3+)
- Regular event schedule
- Community-driven content
- Gradual expansion of #proposals process

## Metrics

### Health Indicators
- New member puzzle completion rate (target: 70%+)
- Messages per day in #limn-only
- Lesson completion progression
- Poetry submissions per week

### Warning Signs
- Puzzle completion < 50%
- Complaints about difficulty
- Limn-only channel going silent
- Moderation escalations increasing
