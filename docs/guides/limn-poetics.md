# Limn Poetics: A Guide to Constraint Poetry

**Author:** Dr. Maren Solvik, Theoretical Linguist
**Date:** 2026-01-25 (Updated 2026-01-30)
**Reference:** linga-land-1t0
**Vocabulary:** v3-natural with collision fixes (`avi` for bird, `bre` for brief, etc.)

---

## 1. Introduction

Limn naturally produces certain poetic structures due to its constraint-based semantics. This guide documents the aesthetic principles that emerge from Limn's design.

---

## 2. Antithesis: The Natural Mode

### 2.1 Why Antithesis Emerges

Limn words define **constraint regions** in semantic space. Antonyms define **complementary regions**:

```
⟦lux⟧ = {light, bright, visible, ...}
⟦nox⟧ = {dark, dim, hidden, ...}
⟦lux⟧ ∩ ⟦nox⟧ = boundary (twilight, ambivalence)
```

Juxtaposing antonyms creates maximum semantic contrast with minimum tokens.

### 2.2 Example Antithetical Structures

**Simple antithesis:**
```limn
lif dea          # life death (mortality)
joy sad          # joy sadness (bittersweet)
beg end          # beginning end (completion)
```

**Parallel antithesis:**
```limn
lif gro | dea dec     # life grows | death decays
joy ris | sad fal     # joy rises | sadness falls
beg ope | end clo     # beginning opens | end closes
```

**Chiasmus (ABBA):**
```limn
lif beg | end dea     # life begins | end death
hot ris | fal col     # heat rises | cold falls
```

## 3. Parallel Structure

### 3.1 Position Creates Correspondence

Without grammar, Limn achieves parallelism through **positional repetition**:

```limn
A B C | D E F | G H I
```

Correspondence:
- Position 1: A, D, G (thematic parallel)
- Position 2: B, E, H (action parallel)
- Position 3: C, F, I (quality parallel)

### 3.2 Semantic Parallelism

Each row should share semantic relationships:

```limn
avi sin | fis swm | per wal
# bird sings | fish swims | person walks
```

Pattern: creature + characteristic-motion

### 3.3 Breaking Parallelism

Deliberate variation creates emphasis:

```limn
lif gro | dea dec | cha end
# life grows | death decays | CHANGE ends
```

The third element breaks the antonym pattern, drawing attention.

## 4. Liminal Expressions

### 4.1 Boundary Poetics

Combining a word with its negation creates **boundary expressions**:

```limn
lux nox          # light-dark = twilight, ambivalence
lif dea          # life-death = mortality, the dying
joy sad          # joy-sadness = bittersweet
```

These point to the boundary between constraint regions.

### 4.2 Paradoxical Beauty

Limn can express paradoxes elegantly:

```limn
ful emp          # full empty (Zen: form is emptiness)
mov sti          # moving still (meditation)
alo tog          # alone together (solitude in community)
```

### 4.3 Liminal Sequences

```limn
beg | beg end | end
# beginning | the transition | ending
```

The middle term captures liminality.

## 5. Intensity and Rhythm

### 5.1 Intensification

Use `ve` (very) for emphasis:

```limn
sad              # sad
ve sad           # very sad
ve ve sad        # intensely sad (rare, emphatic)
```

### 5.2 Weakening

Use `so` (somewhat) for understatement:

```limn
so sad           # somewhat sad, melancholy
so so sad        # barely sad (ironic?)
```

### 5.3 Rhythmic Patterns

Limn poetry can use syllable counts:

**2-3-2 pattern:**
```limn
lux bri | col nox dim | end sti
# bright light | cold dark dim | still end
```

**1-2-1-2 pattern:**
```limn
avi | avi fli | fis | fis swm
# bird | bird flies | fish | fish swims
```

## 6. Negation in Poetry

### 6.1 Creative Negation

`nu` creates "not-X" which includes the boundary:

```limn
nu joy = sad ∪ neutral
nu lif = dea ∪ non-living
```

Poetically:
```limn
nu joy nu sad    # not-joy not-sadness = emotional void
nu lif nu dea    # neither alive nor dead = transcendent?
```

### 6.2 Double Negation

`nu nu X = X` (involution), but can add emphasis:

```limn
nu nu lif        # definitely alive (emphatic)
```

## 7. Scope and Structure

### 7.1 Topic-Comment Structure

Use `|` to create topic-comment:

```limn
nox | lux eme    # darkness: light emerges (dawn)
lif | end com    # life: ending comes (mortality)
```

### 7.2 Parallel Scopes

```limn
mor | wak | wor | ste
# morning: wake | work | step (a day's rhythm)
```

### 7.3 Nested Scopes

```limn
(lif beg) (lif end) | tog
# (life's beginning) (life's end) together = a complete life
```

## 9. Composition Tips

### 9.1 Start with Antonyms

Choose an antonym pair as your thematic core:

```
lif/dea, joy/sad, lux/nox, beg/end, hot/col
```

Build around this opposition.

### 9.2 Use Parallel Position

Align related words in the same position across scopes:

```limn
avi sin | flo dan | win wis
# bird sings | flower dances | wind whispers
```

Position 1: nature element
Position 2: action (personified)

### 9.3 End with Resolution

Traditional structure:
1. Thesis (A)
2. Antithesis (nu A or B)
3. Synthesis (resolution)

```limn
sad alo | joy tog | pea bal
# sad alone | joy together | peace balance
```

### 9.4 Embrace Ambiguity

Limn poetry should invite multiple readings:

```limn
bri fad | lov las
# brightness fades | love lasts
# OR: a fading bride | love lasts
# OR: bright memory fading | love endures
```

---

## 10. Aesthetic Principles Summary

1. **Antithesis is natural** - complementary regions create contrast
2. **Position creates parallelism** - no grammar needed
3. **Boundaries are poetic** - X nu X = liminal space
4. **Brevity is beauty** - constraint intersection is dense
5. **Ambiguity is valued** - multiple readings enrich
