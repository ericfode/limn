WORKING WITH EXPRESSIONS
================================================================================

PARSING EXPRESSIONS:
  1. Extract base word: lon from lon^0.7
  2. Identify operator: ^ (gradient)
  3. Extract parameter: 0.7 (intensity)
  4. Look up meaning in database

COMPOSING NEW EXPRESSIONS:
  1. Select temporal word: pas, fut, gro, etc.
  2. Choose operator:
     - ^ for intensity scaling
     - * for merging concepts
     - @ for connecting
     - without for negation
     - given for conditioning
     - ± for duality
  3. Apply operator: pas^0.8 or pas*fut
  4. Nest if needed: (pas*fut)*cha

FINDING EQUIVALENT EXPRESSIONS:
  - Look for expressions with similar meanings
  - Group by semantic domain
  - Check gradient variations (0.1 vs 0.9)
  - Explore fusion combinations
